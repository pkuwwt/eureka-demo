
# Spring Boot Hello World


## Compile

Directly run

```bash
mvn spring-boot:run
```

Test
```bash
mvn test
```

Compile to jar

```bash
mvn package
java -jar target/spring-boot-*.jar
```

Browse `http://127.0.0.1:8080` and `http://127.0.0.1:8080/actuator` for status.
